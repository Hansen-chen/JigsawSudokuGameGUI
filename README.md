# JigsawSudokuGameGUI

## Note

0. Do not put invalid board txt file in board folder of this project

## Run in Mac/windows

1. under project root folder, run "cabal install .." necessary packages
2. stack install
3. stack exec JigsawSudokuGameGUI-exe

## Fast Build for Debug in windows

1. under project root folder, run "cabal install .." necessary packages
2. "stack ghci"
3. "main"
